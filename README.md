# moro (모로)
### 컬러워크로 발견한 일상 속 색을 나만의 팔레트로 기록하고 공유하는 SNS 앱
moro는 사진 속 색을 중심으로 기록하고 소통하는 컬러 기반 SNS 앱입니다.
사용자가 사진을 촬영하거나 업로드하면 앱이 자동으로 주요 색상을 추출하고, 이 색을 기반으로 피드, 지도, 친구 추천, 미션 등 다양한 기능이 연결됩니다.
색을 재료로 취향을 표현하고, 색을 통해 새로운 경험과 사람을 발견하는 컬러워크(Color Work) 플랫폼을 지향합니다.

이 리포지토리는 **Android(Kotlin) + Jetpack Compose 기반의 프론트엔드 앱**을 위한 공간입니다.  
백엔드 서버는 별도의 리포지토리에서 관리됩니다.

---

## 🚀 Tech Stack
- **Kotlin**
- **Android Studio**
- **Jetpack Compose**
- **Coroutines / Flow**
- **CameraX** (사진 촬영)

---

## ▶️ Getting Started
1. 이 리포를 클론합니다.
2. Android Studio에서 열기
3. 필요한 SDK 설치 후 Gradle Sync
4. 로컬 환경 설정 (필요 시 `local.properties` 또는 `/config` 파일)
5. 앱 빌드 및 실행
